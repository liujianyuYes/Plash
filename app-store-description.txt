把任意网站变成你的桌面壁纸。

Plash 可以让你拥有高度动态的桌面壁纸。你可以显示喜欢的新闻网站、社交动态，或随机的风景照片。使用场景几乎没有限制，甚至可以把动画 GIF 设置为壁纸。你还可以添加多个网站，并在它们之间轻松切换。


■ 功能

- 显示远程或本地网站
- 与网站交互（“浏览模式”）
- 按自定义间隔自动重新加载网站
- 添加多个网站
- 在一个显示器或所有已连接的显示器上显示网站
- 反转网站颜色（伪深色模式）
- 为网站添加自定义 CSS 和 JavaScript
- 降低不透明度
- 透明背景
	如果网站没有背景色，系统壁纸会透出来。
- 使用电池时自动停用
- 默认静音
- 单张图片会按比例填满屏幕
- 隐藏菜单栏图标
- 支持快捷指令
- 支持脚本控制（请参阅 Plash 官网）
- 分享扩展

注意：Plash 现在可以在单个显示器或所有已连接的显示器上显示同一个网站。为每个显示器单独设置不同网站的功能仍在计划中。

■ 使用技巧

‣ 浏览模式

启用“浏览模式”后，你可以与网站交互。在此模式下，可以右键返回/前进、重新加载，并放大页面内容（缩放级别会被保存）。你也可以通过双指捏合放大。这不同于普通页面缩放，它会放大页面中的特定区域，而不是简单放大所有内容。

浏览模式启用时，Plash 会向 “html” 元素注入名为 “plash-is-browsing-mode” 的 CSS 类。你可以使用这个类为浏览模式定制网站样式。

如果点击链接会在新窗口中打开，你可以按住 Option 键点击链接，让它在 Plash 主窗口中打开。

‣ 放大网站

要放大网站，请启用“浏览模式”，右键点击网站，然后选择“放大”。

‣ 用于屏幕宽高的 URL 占位符

你可以在任意 URL 中使用 `[[screenWidth]]` 和 `[[screenHeight]]`，Plash 会自动替换为正确的值。例如：`https://source.unsplash.com/random/[[screenWidth]]x[[screenHeight]]?puppy`。

■ 常见问题

‣ 如何用键盘快捷键切换到指定网站？

在“快捷指令”App 中创建一个使用“设置当前网站”动作的快捷指令，然后为该快捷指令设置键盘快捷键。
